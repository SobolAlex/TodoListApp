# TodoListApp

Консольное приложение на C# для управления списком задач (Todo List).

---

# Функционал
- Добавление задач с автоматическим номером
- Редактирование задачи
- Возможность отметки задачи(выполнено/не выполнено)
- Удаление выбранной задачи
- Удаление всех задач сразу
- Сохранение и загрузка задач из файла
- Простой консольный интерфейс

---

# Технологии
- C#  
- .NET 8 
- Консольное приложение  
- Git для контроля версий
